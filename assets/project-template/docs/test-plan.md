# Test plan

## First power

- [ ] Visual inspection and continuity checks.
- [ ] Current-limited bench supply.
- [ ] Rails measured with no optional loads.
- [ ] No unexpected heating or backfeed.

## Worst-case power and thermal

- [ ] All permitted simultaneous loads enabled.
- [ ] Continuous and peak current recorded.
- [ ] Critical component temperatures recorded.
- [ ] Test repeated at representative enclosure temperature.

## Functional

- [ ] Boot/reset safe states.
- [ ] Inputs and sensors.
- [ ] Relays/actuators and flyback behavior.
- [ ] Communications with real cable/termination.
- [ ] External peripherals and connector pinout.
- [ ] ADC/calibration and threshold hysteresis.

## Reliability

- [ ] Power-cycle test.
- [ ] Network/communication loss and recovery.
- [ ] Brownout/reset behavior.
- [ ] Burn-in duration and conditions recorded.

## Results

Store dated results and evidence below `docs/validation/` and link them here.
